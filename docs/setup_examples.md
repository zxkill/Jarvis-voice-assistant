# Примеры настройки

Этот документ содержит типовые примеры конфигурации Jarvis-Pi.

## Минимальная конфигурация

```ini
[LLM]
light_model = llama2
heavy_model = llama2:13b
```

## Включение M5Stack

```ini
[DISPLAY]
driver = serial
```
